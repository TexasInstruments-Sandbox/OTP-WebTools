# TI SCI Web Tools

Web-based utilities for Texas Instruments System Control Interface debugging and development.

## Tools

### OTP Key Writer Error Code Decoder
Decode OTP Key Writer error codes from decimal, hexadecimal, or binary format. Provides detailed descriptions for all 32 error code bits.

[Documentation](https://software-dl.ti.com/tisci/esd/latest/2_tisci_msgs/security/keywriter.html)
